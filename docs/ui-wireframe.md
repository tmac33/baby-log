# UI Wireframe (Draft)

## 1. Home (Today)
- Header: Baby name + Today date
- Quick Actions: [Feed] [Diaper] [Voice]
- Today Summary cards:
  - Last feed time + amount
  - Last diaper time + type
- Timeline list (today)

## 2. Add Feed
- Amount (ml) input
- Time picker (default now)
- Note
- Save

## 3. Add Diaper
- Type: Wet / Dirty / Mixed
- Time picker
- Note
- Save

## 4. Voice Capture
- Microphone button (hold/tap)
- Live transcript
- Parsed result card
- Confirm / Edit / Cancel

## 5. History
- Filter by day/week
- Export (future)
